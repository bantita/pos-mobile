## 1. เริ่มต้นใช้งาน

### ความต้องการของระบบ
- **Mobile**: iOS 14+ / Android 10+ (Expo Go หรือ Build)
- **Web**: Chrome 90+, Firefox 90+, Safari 14+, Edge 90+
- **หน้าจอ**: รองรับ 375px (mobile) ถึง 2560px (ultra-wide)

### วิธีเปิดใช้งาน
```bash
# Development
npm start          # Expo Dev Server
npm run web        # เปิด Web version ที่ localhost:8081

# Production Build
eas build --platform android
eas build --platform ios
npx expo export:web
```

## 2. สมัครร้านค้า

### Flow
```
หน้า Welcome → กดปุ่ม "สมัครร้านค้าใหม่"
→ กรอกข้อมูลร้าน (ชื่อ, ประเภทธุรกิจ, ขนาด)
→ กรอกข้อมูลเจ้าของ (ชื่อ, เบอร์โทร, อีเมล)
→ ตั้งรหัสผ่าน
→ เข้าสู่ Dashboard
```

### ข้อมูลที่ต้องกรอก
| ฟิลด์ | บังคับ | ตัวอย่าง |
|--------|--------|----------|
| ชื่อร้าน | ✅ | ร้านสะดวกซื้อ ABC |
| ประเภทธุรกิจ | ✅ | ร้านค้าปลีก / ร้านอาหาร / ร้านบริการ |
| ขนาดธุรกิจ | ✅ | เล็ก (1 สาขา) / กลาง / ใหญ่ |
| ชื่อเจ้าของ | ✅ | สมชาย ใจดี |
| เบอร์โทร | ✅ | 081-234-5678 |
| อีเมล | ❌ | owner@shop.com |
| รหัสผ่าน | ✅ | (ขั้นต่ำ 6 ตัวอักษร) |

### หน้าจอ
- หน้า Welcome มีปุ่ม "เข้าสู่ระบบ" และ "สมัครร้านค้าใหม่"
- ฟอร์มสมัครแบ่งเป็นขั้นตอน (Step wizard)
- เลือกประเภทร้านจะกำหนด feature ที่เปิดใช้ (เช่น ร้านบริการจะมี "จัดการช่าง")

## 5. ขายสินค้า (POS)

### Flow หลัก
```
เปิดหน้า POS
→ สแกนบาร์โค้ด หรือ เลือกสินค้าจาก Grid/ค้นหา
→ สินค้าเข้าตะกร้า (ปรับจำนวน, ให้ส่วนลด, เปลี่ยนราคา)
→ เลือกสมาชิก (ถ้ามี)
→ กรอกคูปอง / ใช้คะแนน (ถ้ามี)
→ ส่วนลดท้ายบิล (ถ้ามี)
→ กดชำระเงิน (เงินสด / QR / โอน / บัตร)
→ พิมพ์ใบเสร็จ
→ คะแนนสะสมอัตโนมัติ (ถ้าเลือกสมาชิก)
```

### ฟีเจอร์ย่อย
| ฟีเจอร์ | คำอธิบาย |
|---------|----------|
| Product Grid | แสดงสินค้าเป็น Grid + filter ตาม Category |
| Barcode Scan | พิมพ์/สแกนบาร์โค้ดในช่องค้นหา |
| ส่วนลดรายการ | กดที่สินค้าในตะกร้า → ใส่ % หรือ บาท |
| เปลี่ยนราคา | กดที่สินค้า → แก้ราคาขาย (เฉพาะบิลนี้) |
| ส่วนลดท้ายบิล | กดที่ยอดรวม → ใส่ % หรือ บาท |
| เลือกสมาชิก | กดปุ่ม "เลือกสมาชิก" → ค้นหาชื่อ/เบอร์/เลขสมาชิก |
| ใช้คูปอง | กรอกรหัสคูปอง → ตรวจสอบ → หักส่วนลด |
| ใช้คะแนน | เลือกจำนวนคะแนนที่จะใช้ → แปลงเป็นส่วนลด |
| พักบิล (Hold) | พักบิลปัจจุบัน → เปิดบิลใหม่ → เรียกคืนภายหลัง |
| ช่องทางชำระ | เงินสด (คำนวณทอน), QR, โอน, บัตรเครดิต |
| ใบกำกับภาษี | กดออกใบกำกับภาษีเต็มรูปแบบ |
| จอที่ 2 | แสดงรายการให้ลูกค้าเห็น (Customer Display) |

## 6. ประวัติขาย

### Flow
```
หน้าประวัติขาย → เห็น Listing บิลทั้งหมด
→ Filter ตาม Tab (ทั้งหมด / ขายสำเร็จ / คืน-ยกเลิก)
→ ค้นหาเลขบิล / สมาชิก / พนักงาน
→ กดบิล → ดูรายละเอียด (สินค้า, ยอดเงิน, ชำระ, สมาชิก)
→ Actions: พิมพ์ใบเสร็จ / ออกใบกำกับภาษี / คืนสินค้า
```

### ข้อมูลในรายละเอียดบิล
- เลขบิล, วัน/เวลา, พนักงาน, POS
- รายการสินค้า: ชื่อ, จำนวน, ราคา, ส่วนลด, รวม
- สรุปยอด: subtotal, ส่วนลด, คูปอง, คะแนน, VAT, ยอดสุทธิ
- สมาชิก (ถ้ามี): ชื่อ, เลขสมาชิก, คะแนนที่ได้
- ช่องทางชำระ, รับเงิน, ทอน

## 7. สินค้า

### Tabs
| Tab | คำอธิบาย |
|-----|----------|
| รายการสินค้า | CRUD สินค้า, Multi-UOM, Multi-Barcode, รูปภาพ |
| กำหนดราคา | ราคาทุน/ขาย/สมาชิก, ดูกำไร% |
| คลังสินค้า | สต๊อกปัจจุบัน, สถานะ (ปกติ/ใกล้หมด/หมด), KPI |
| Import/Export | นำเข้า Excel/CSV, ส่งออกข้อมูล |
| มาสเตอร์ | หน่วยนับ, แบรนด์, หมวดหมู่ |

### Flow เพิ่มสินค้า
```
กดปุ่ม "+ เพิ่มสินค้า"
→ กรอก: รหัส, บาร์โค้ด, ชื่อ, หมวดหมู่, หน่วยฐาน
→ ตั้งราคาทุน + ราคาขาย
→ เพิ่มหน่วยนับ (ถ้ามี เช่น แพ็ค = 12 ชิ้น)
→ เพิ่มบาร์โค้ดต่อหน่วย
→ อัพโหลดรูป (ถ้าต้องการ)
→ บันทึก
```

## 9. CRM สมาชิก

### Sub-menus
| เมนู | คำอธิบาย |
|------|----------|
| ข้อมูลสมาชิก | ค้นหา, ดูรายละเอียด, แก้ไข, ระงับ/ลบ |
| เพิ่มสมาชิก | กรอก ชื่อ/เบอร์/วันเกิด/ระดับ |
| ระดับสมาชิก | Silver/Gold/Platinum — เงื่อนไข, discount, multiplier |
| ตั้งค่าคะแนน | อัตราสะสม, อัตราแลก, ขั้นต่ำ, หมดอายุ |
| ปรับปรุงคะแนน | เพิ่ม/ลด manual |
| คูปอง / Voucher | สร้าง/จัดการ (เชื่อมกับ Promotion Store), ส่งผ่าน LINE/SMS |
| Campaign | สร้างแคมเปญส่งข้อความ |
| Communication | LINE Broadcast / SMS / Push (stats, quick send, templates) |
| Gamification | วงล้อนำโชค, สุ่มรางวัล, Check-in |
| Segment ลูกค้า | กลุ่มตามเงื่อนไข (ซื้อ 30 วัน, ยอดสะสม, ระดับ) |
| ประวัติการซื้อ | ดูบิลของสมาชิก |
| Wallet / Credit | เติมเงิน, ชำระจาก wallet |
| รายงาน CRM | สรุปสมาชิก, คะแนน, retention |
| ตั้งค่า CRM | การตั้งค่าทั่วไป |

### Flow เพิ่มสมาชิก
```
กดเมนู "เพิ่มสมาชิก"
→ กรอก: เบอร์โทร (ตรวจซ้ำ), ชื่อ, วันเกิด
→ เลือกระดับเริ่มต้น (default: Member)
→ บันทึก → ได้เลขสมาชิก MEM-XXXXXX
```

## 10. โปรโมชั่น

### ประเภทโปรโมชั่น
| ประเภท | ตัวอย่าง |
|--------|----------|
| ลดเปอร์เซ็นต์ | ลด 10% ทุกสินค้า |
| ลดจำนวนเงิน | ลด 50 บาท เมื่อซื้อ 500+ |
| คูปอง | SUMMER2024 ลด 100 บาท (200 สิทธิ์) |
| ราคาสมาชิก | Gold ลด 5% |
| ซื้อ X แถม Y | ซื้อ 3 แถม 1 |
| Happy Hour | 17:00-19:00 ลด 20% เครื่องดื่ม |

### Flow สร้างโปร
```
กด "+ สร้างโปรโมชั่น"
→ เลือกประเภท → กรอกเงื่อนไข
→ กำหนดวันเริ่ม/สิ้นสุด, สินค้า/หมวดที่ใช้ได้
→ Stacking Rules (ซ้อนกับโปรอื่นได้ไหม)
→ บันทึก → สถานะ Active
→ ระบบคำนวณอัตโนมัติเมื่อขายสินค้า
```
