# 06 — ตั้งค่าระบบ

## เมนูตั้งค่าทั้งหมด

ไปที่ Tab **"ตั้งค่า"** จะพบหมวดต่างๆ:

---

## 🏪 ข้อมูลร้านค้า

| ค่า | คำอธิบาย |
|-----|----------|
| ชื่อร้าน | แสดงในใบเสร็จ / header |

| ที่อยู่ | พิมพ์ในใบเสร็จ |
| เบอร์โทร | แสดงในใบเสร็จ |

| ประเภทร้าน | ค้าปลีก / ร้านอาหาร / ร้านบริการ / อื่นๆ |

---

## 🏢 สาขา & จุดขาย (POS)

### สาขา
- สร้างสาขา (ชื่อ, ที่อยู่, ผู้จัดการ)
- เปิด/ปิดสาขา
- กำหนดคลังสินค้าของสาขา

### POS (จุดขาย)
- สร้าง POS ในแต่ละสาขา (เช่น "POS 1 เคาน์เตอร์หน้า")
- กำหนด:
  - ชื่อ POS
  - เครื่องพิมพ์ที่เชื่อม
  - ช่องทางชำระที่เปิด
  - พนักงานที่ใช้ได้

---

## 💳 ช่องทางชำระเงิน

เปิด/ปิดช่องทาง + ตั้งค่าแต่ละอัน:

| ช่องทาง | ตั้งค่า |
|---------|--------|
| เงินสด | เปิด/ปิด |
| PromptPay | เลข PromptPay (เบอร์/Tax ID) |
| บัตรเครดิต | เชื่อม Payment Gateway |
| โอนเงิน | ชื่อบัญชี + เลขบัญชี + ธนาคาร |
| E-Wallet | เชื่อม TrueMoney / LINE Pay API |

## 🖨️ เครื่องพิมพ์

### ประเภทที่รองรับ
- **Thermal Printer** (ใบเสร็จ 58mm/80mm) — Bluetooth/USB/Network
- **PDF Print** (พิมพ์ผ่าน browser/system print)

### ตั้งค่า
1. ตั้งค่า > เครื่องพิมพ์ > "+ เพิ่ม"
2. เลือกประเภท (Bluetooth / USB / Network)
3. Pair / กรอก IP
4. ทดสอบพิมพ์

### Template ใบเสร็จ
- เลือก layout (แบบย่อ / แบบเต็ม / ใบกำกับภาษี)
- ปรับ: แสดง/ซ่อน โลโก้, QR code, ข้อความท้ายใบเสร็จ
- Preview ก่อนใช้งาน

### ตั้งค่าการพิมพ์
| ค่า | คำอธิบาย |
|-----|----------|
| พิมพ์อัตโนมัติ | พิมพ์ทันทีหลังชำระ |
| จำนวนสำเนา | 1-3 ใบ |
| เปิดลิ้นชักเงิน | เปิดอัตโนมัติเมื่อรับเงินสด |
| ขนาดกระดาษ | 58mm / 80mm |

## 👥 จัดการพนักงาน

### เพิ่มพนักงาน
1. ตั้งค่า > พนักงาน > "+ เพิ่ม"
2. กรอก: ชื่อ, เบอร์โทร (เป็น username), PIN 4 หลัก
3. เลือก Role (Owner/Manager/Cashier/Stock)
4. กำหนดสาขา + POS ที่ใช้ได้

### PIN สำหรับ action พิเศษ
- ยกเลิกบิล → ต้องใส่ PIN (Manager ขึ้นไป)
- เปลี่ยนราคา → ต้องใส่ PIN
- Reprint ใบเสร็จ → ต้องใส่ PIN
- ตั้งค่าได้ว่า action ไหนต้องใช้ PIN

## 🔐 สิทธิ์ & Role

### Permission Matrix
ดูและแก้ไขสิทธิ์ทุก role ในตารางเดียว:

| Module | Owner | Manager | Cashier | Stock |
|--------|-------|---------|---------|-------|
| ขาย | ✓ | ✓ | ✓ | ✗ |
| ยกเลิกบิล | ✓ | ✓ | ✗ | ✗ |
| สินค้า | ✓ | ✓ | ✗ | ✗ |
| คลัง | ✓ | ✓ | ✗ | ✓ |
| รายงาน | ✓ | ✓ | ✗ | ✗ |
| ตั้งค่า | ✓ | บางส่วน | ✗ | ✗ |
| CRM | ✓ | ✓ | ดูอย่างเดียว | ✗ |

### Custom Role
- สร้าง Role ใหม่ได้ (เช่น "หัวหน้าแคชเชียร์")
- กำหนด permission แต่ละ module

## 🔒 ความปลอดภัย

| ค่า | คำอธิบาย |
|-----|----------|
| Session Timeout | ล็อกอัตโนมัติหลังไม่ใช้ (5/10/15/30 นาที) |
| PIN Length | 4 หรือ 6 หลัก |
| Login attempts | ล็อก 5 นาทีหลังผิด 5 ครั้ง |
| Force password change | บังคับเปลี่ยนทุก 90 วัน |
| Device management | ดู/ถอดอุปกรณ์ที่ login อยู่ |

## 📺 จอลูกค้า (Customer Display)

### เปิดใช้
1. ตั้งค่า > จอลูกค้า > เปิดใช้งาน
2. เปิด browser อีกหน้าต่าง → ลากไปจอที่ 2 (HDMI)
3. URL: `https://your-pos.com/?display=1`

### ตั้งค่า
- แสดง/ซ่อน: ยอดรวม real-time, โลโก้ร้าน, โฆษณา
- จัดการ slideshow โฆษณา (รูป/วิดีโอ)
- แสดงโปรโมชั่นที่กำลัง apply

---

## 📡 ERP Integration

ถ้าต้องการเชื่อมกับระบบ ERP (SAP, Oracle, Microsoft Dynamics):
1. ตั้งค่า > ERP Integration
2. กรอก API endpoint ของ ERP
3. map fields (สินค้า, ยอดขาย, สต๊อก)
4. ตั้ง sync frequency (real-time / ทุกชั่วโมง / สิ้นวัน)

---

## 🔍 Audit Log

ดูทุก action ที่เกิดขึ้นในระบบ:
- ใครทำ / ทำอะไร / เมื่อไหร่ / จากอุปกรณ์ไหน
- filter ตาม: module, action, user, วันที่
- export ได้ (CSV/PDF)
