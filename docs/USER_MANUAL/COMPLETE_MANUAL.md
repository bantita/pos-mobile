## Quick Start (เริ่มใช้งานใน 5 นาที)

1. เปิดแอป → **สมัคร** (เบอร์ + ชื่อร้าน + รหัส)
2. **เพิ่มสินค้า** 3-5 รายการ (หรือ import CSV)
3. ไปหน้า **ขาย** → กดสินค้า → **ชำระเงิน** → เสร็จ!
4. (Optional) เปิด **PromptPay** ที่ตั้งค่า > ชำระเงิน
5. (Optional) เพิ่ม **สมาชิก** เพื่อเก็บคะแนน
