# 02 — การขาย POS

## หน้าจอขายหลัก

เข้าที่ Tab **"ขาย"** จะพบ:
- ปุ่มหมวดหมู่สินค้า (ด้านบน)
- Grid สินค้า (กดเพื่อเพิ่มลงตะกร้า)
- ช่องค้นหา / สแกนบาร์โค้ด
- ตะกร้า (ด้านขวาบน Web / ด้านล่าง Mobile)

---

## วิธีขายสินค้า

### 1. เพิ่มสินค้าลงตะกร้า
- **กดที่สินค้า** ใน Grid → เพิ่ม 1 ชิ้น
- **กดค้าง** → เลือกจำนวน / เลือก UOM (เช่น ขายเป็นแพ็ค)
- **สแกนบาร์โค้ด** → ระบบค้นหาสินค้าอัตโนมัติ
- **พิมพ์ชื่อ/รหัส** ในช่องค้นหา

### 2. แก้ไขรายการในตะกร้า
- กด **+/-** เพื่อเพิ่ม/ลดจำนวน
- กด **ถังขยะ** เพื่อลบรายการ
- กดที่รายการ → แก้ไขราคา / ส่วนลดเฉพาะรายการ

### 3. ใส่ส่วนลด / คูปอง
- กด **"คูปอง"** → กรอกรหัสคูปอง → ระบบคำนวณส่วนลดให้
- กด **"ส่วนลด"** → ใส่ % หรือ จำนวนเงิน

### 4. เลือกสมาชิก
- กด **"สมาชิก"** → ค้นหาเบอร์โทร/ชื่อ → เลือก
- ระบบจะ:
  - แสดงคะแนนคงเหลือ
  - คำนวณราคาสมาชิก (ถ้ามี)
  - เตรียมให้คะแนนหลังชำระ

### 5. แลกคะแนน
- หลังเลือกสมาชิก กด **"ใช้คะแนน"**
- กรอกจำนวนคะแนน → ระบบคำนวณส่วนลดให้
- ขั้นต่ำ 100 คะแนน = 1 บาท (ค่า default, ปรับได้)

---

## ชำระเงิน

กด **"ชำระเงิน"** → เข้าหน้า Payment:

### ช่องทางชำระ
| ช่องทาง | วิธีใช้ |
|---------|--------|
| 💵 เงินสด | กรอกจำนวนเงินที่รับ → แสดงเงินทอน |
| 📱 QR Code | แสดง PromptPay QR → ลูกค้าสแกนจ่าย → กด "ยืนยัน" |
| 💳 บัตรเครดิต | เสียบ/แตะบัตร → รอ approve |
| 🏦 โอนเงิน | แสดง QR/เลขบัญชี → ลูกค้าโอน → กด "ยืนยัน" |
| 📲 E-Wallet | เปิด app wallet → สแกน → รอ status |

### ชำระแยก (Split Payment)
- เลือกหลายช่องทางได้ (เช่น เงินสด 200 + QR 150)
- ระบบคำนวณยอดคงเหลือให้อัตโนมัติ

---

## หลังชำระสำเร็จ

1. **พิมพ์ใบเสร็จ** (อัตโนมัติ ถ้าตั้งค่าไว้)
2. **ส่งใบเสร็จทาง LINE** (ถ้าสมาชิกลิงก์ LINE ไว้)
3. **ให้คะแนนสมาชิก** (อัตโนมัติ ตามอัตราที่ตั้ง)
4. **แจ้งเตือน LINE** "ได้รับ X คะแนน" (ถ้าเปิดใช้)
5. **เปิดบิลใหม่** ขายต่อได้ทันที

---

## โหมดการขาย

| โหมด | คำอธิบาย |
|------|----------|
| ปกติ | ขายปกติ ลูกค้าซื้อทันที |
| Pre-order | สั่งจอง — ยังไม่ตัดสต๊อก |
| Delivery | จัดส่ง — บันทึกที่อยู่ |
| Dine-in | ทานที่ร้าน — ระบุโต๊ะ |

---

## สแกนบาร์โค้ด

- กดไอคอน **📷 Barcode** → เปิดกล้อง
- หรือใช้ **เครื่องสแกนภายนอก** (Bluetooth/USB) — พิมพ์ตรงๆ ในช่องค้นหา
- รองรับ: EAN-13, EAN-8, Code 128, Code 39, QR Code

---

## ยกเลิกบิล / คืนสินค้า

### ยกเลิกบิล (Void)
- ไปที่ **ประวัติการขาย** → เลือกบิล → กด **"ยกเลิก"**
- ต้องกรอกเหตุผล + ใส่ PIN (ถ้าตั้งค่าไว้)
- สต๊อกจะถูกคืนอัตโนมัติ

### คืนสินค้า (Return)
- ไปที่ **ประวัติ** → เลือกบิล → กด **"คืนสินค้า"**
- เลือกรายการที่คืน + จำนวน
- เลือกวิธีคืนเงิน (เงินสด / โอน / คืนเป็นคะแนน)
