# 09 — ประวัติ, ทีมงาน & Sync

## ประวัติการขาย

### ดูประวัติ
- Tab "ขาย" → "ประวัติ" หรือ เมนู "ประวัติ" บน Dashboard
- แสดง: เลขบิล, วันที่, ยอด, พนักงาน, สถานะ
- filter: วันที่, ช่องทางชำระ, สมาชิก, สถานะ
- กดดูรายละเอียด: รายการสินค้า, การชำระ, ส่วนลด, สมาชิก

### Reprint ใบเสร็จ
- เปิดบิล → กด "พิมพ์ซ้ำ"
- ต้องใส่ PIN (ถ้าตั้งค่าไว้)

---

## กะเงิน (Shift)

### เปิดกะ
1. เข้าหน้าขาย → ระบบถามให้เปิดกะ
2. กรอก **ยอดเงินเปิดกะ** (เงินทอนเริ่มต้น)
3. เริ่มขายได้

### ปิดกะ
1. กด **"ปิดกะ"**
2. นับเงินในลิ้นชัก → กรอก **ยอดเงินปิดกะ**
3. ระบบเปรียบเทียบ:
   - ยอดที่ควรมี (คำนวณจากบิล)
   - ยอดที่นับได้ (ที่กรอก)
   - ส่วนต่าง (ขาด/เกิน)
4. แสดง **สรุปกะ**: จำนวนบิล, ยอดขาย, แยกช่องทาง

## จัดการทีมงาน

### ดู list พนักงาน
ตั้งค่า > พนักงาน:
- ชื่อ, เบอร์, Role, สาขา, POS, สถานะ
- ดู last login, จำนวนครั้ง login

### เพิ่มพนักงาน
1. กด "+ เพิ่ม"
2. กรอก: ชื่อ, เบอร์โทร (เป็น username), PIN
3. เลือก Role
4. กำหนดสาขา/POS

### ระงับ/เปิดใช้
- กด "ระงับ" → พนักงาน login ไม่ได้
- กด "เปิดใช้" → กลับมาใช้ได้

### Reset PIN
- Owner/Manager กด "Reset PIN" → กำหนด PIN ใหม่

## 🔄 Sync & Offline

### การทำงาน Offline
- ระบบทำงานได้แม้ไม่มี internet
- ขาย, เพิ่มสมาชิก, รับ/เบิกสินค้า → บันทึกใน local
- เมื่อ online → sync อัตโนมัติ

### Sync Queue
Tab "Sync" แสดง:
- **Pending** — รอส่งขึ้น server
- **Syncing** — กำลังส่ง
- **Success** — ส่งสำเร็จ
- **Failed** — ส่งไม่สำเร็จ (กด retry ได้)
- **Conflict** — ข้อมูลขัดแย้ง (ต้อง resolve)

### Conflict Resolution
เมื่อข้อมูลขัดแย้ง (เช่น แก้ราคาจาก POS 1 และ Back Office พร้อมกัน):
1. ระบบแสดง:
   - **ค่าจาก Client** (POS ที่ใช้อยู่)
   - **ค่าจาก Server** (จาก Back Office / POS อื่น)
2. เลือก:
   - **ใช้ของ Client** — ใช้ค่าจาก POS นี้
   - **ใช้ของ Server** — ใช้ค่าจาก server
   - **แก้ไขเอง** — กรอกค่าที่ถูกต้อง

### Sync Monitor (สำหรับ Admin)
ตั้งค่า > Sync Monitor:
- ดูทุกอุปกรณ์ในร้าน
- สถานะ online/offline
- จำนวน pending ของแต่ละเครื่อง
- last sync time
- Force sync ได้
