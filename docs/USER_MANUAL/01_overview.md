# 01 — ภาพรวมระบบ POS CRM

## ระบบ POS CRM คืออะไร?

ระบบ POS CRM เป็นแอปพลิเคชันจุดขาย (Point of Sale) พร้อมระบบจัดการลูกค้าสัมพันธ์ (Customer Relationship Management) ในตัว ออกแบบมาสำหรับร้านค้าปลีก/ร้านบริการ รองรับทั้ง:

- **Mobile** (iOS/Android) — ใช้เป็น POS เคลื่อนที่
- **Web** (Browser) — ใช้เป็น Back Office / POS บนเครื่อง PC
- **Kiosk Mode** — ล็อกหน้าจอเป็น POS สำหรับพนักงาน
- **Customer Display** — แสดงจอที่ 2 ให้ลูกค้าเห็นรายการ

## บทบาทผู้ใช้ (Roles)

| Role | สิทธิ์ |
|------|--------|
| **Owner (เจ้าของ)** | ทำได้ทุกอย่าง, ดูรายงาน, ตั้งค่าร้าน |
| **Manager (ผู้จัดการ)** | ขาย, จัดการสินค้า/สต๊อก, ดูรายงาน, จัดการพนักงาน |
| **Cashier (แคชเชียร์)** | ขายเท่านั้น, ดูประวัติตัวเอง |
| **Stock Staff (คลัง)** | จัดการสต๊อก, รับ/เบิกสินค้า |
| **Report Viewer** | ดูรายงานอย่างเดียว |
| **Admin (Super Admin)** | เข้าถึงได้ทุกฟังก์ชัน รวม module toggles |

## การเข้าใช้งานครั้งแรก

### สมัครใช้งาน (Register)
1. เปิดแอป → กด **"สมัครใช้งาน"**
2. กรอก:
   - เบอร์โทรศัพท์ (ใช้เป็น username)
   - ชื่อ
   - รหัสผ่าน (อย่างน้อย 4 ตัว)
   - ชื่อร้านค้า
3. กด **"สมัคร"** → ระบบสร้างร้านใหม่ให้อัตโนมัติ
4. เข้าสู่หน้า Dashboard ได้ทันที

### Login
- กรอกเบอร์โทร + รหัสผ่าน
- หรือใช้ **OTP** (ระบบส่ง SMS มาให้)
- Demo: `admin` / `1234` (เจ้าของร้าน)

## การตั้งค่าเริ่มต้น (แนะนำ)

หลัง Login ครั้งแรก ควรตั้งค่าตามลำดับนี้:

1. **ตั้งค่า > ข้อมูลร้าน** — ชื่อร้าน, โลโก้, ที่อยู่
2. **ตั้งค่า > สาขา & POS** — สร้างสาขา, กำหนด POS
3. **สินค้า** — เพิ่มสินค้า (หรือ import CSV)
4. **ตั้งค่า > ช่องทางชำระ** — เปิดเงินสด/QR/บัตร
5. **สมาชิก** — เปิดระบบ CRM (optional)
6. **โปรโมชั่น** — สร้างโปรแรก (optional)
