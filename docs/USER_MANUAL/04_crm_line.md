# 04 — CRM & LINE OA

## ระบบ CRM คืออะไร?

ระบบจัดการลูกค้าสัมพันธ์ ประกอบด้วย:
- **สมาชิก** — ลงทะเบียน, เก็บข้อมูล, ประวัติซื้อ
- **คะแนนสะสม (Loyalty Points)** — ให้คะแนนเมื่อซื้อ, แลกส่วนลด
- **ระดับสมาชิก (Tiers)** — Member → Silver → Gold → Platinum → VIP
- **LINE OA** — ส่งข้อความ/คูปอง/แจ้งเตือนผ่าน LINE

---

## สมาชิก

### เพิ่มสมาชิก
1. ไปที่ Tab **"สมาชิก"** → กด **"+ เพิ่มสมาชิก"**
2. กรอก:
   - เบอร์โทร *(จำเป็น)*
   - ชื่อ *(จำเป็น)*
   - วันเกิด *(เพื่อโปรวันเกิด)*
   - ระดับเริ่มต้น (ปกติ = Member)
3. กด **"บันทึก"**

### เพิ่มสมาชิกตอนขาย
- ตอนอยู่หน้าขาย → กด **"สมาชิก"** → กด **"+ สร้างใหม่"**
- กรอกแค่เบอร์ + ชื่อ → สร้างทันทีไม่ต้องออกจากหน้าขาย

### ค้นหาสมาชิก
- พิมพ์ **ชื่อ / เบอร์โทร / เลขสมาชิก** ในช่องค้นหา
- ระบบแสดงผลแบบ real-time ขณะพิมพ์

---

## คะแนนสะสม (Loyalty Points)

### ตั้งค่าอัตราคะแนน
ไปที่ **ตั้งค่า > CRM > คะแนนสะสม:**

| ค่า | ความหมาย | ตัวอย่าง |
|-----|----------|----------|
| อัตราให้คะแนน | ทุก X บาท ได้ 1 คะแนน | 25 บาท = 1 คะแนน |
| อัตราแลกคะแนน | 1 คะแนน = X บาท | 1 คะแนน = 1 บาท |
| ขั้นต่ำแลก | แลกอย่างน้อยกี่คะแนน | 100 คะแนน |
| วันหมดอายุ | คะแนนหมดอายุกี่วัน | 365 วัน |

### การให้คะแนน (อัตโนมัติ)
- เมื่อชำระเงินสำเร็จ + เลือกสมาชิกไว้
- ระบบคำนวณคะแนนอัตโนมัติ (เช่น ซื้อ 250 บาท ÷ 25 = 10 คะแนน)
- แจ้งลูกค้าผ่าน LINE (ถ้าเชื่อมไว้)

### การแลกคะแนน
1. หน้าขาย → เลือกสมาชิก → กด **"ใช้คะแนน"**
2. กรอกจำนวนคะแนน → ระบบแสดงส่วนลด
3. ชำระเงิน → คะแนนถูกหักอัตโนมัติ

---

## ระดับสมาชิก (Member Levels)

### ตั้งค่าเงื่อนไข
ไปที่ **ตั้งค่า > CRM > ระดับสมาชิก:**

| ระดับ | เงื่อนไข (ยอดสะสม) | สิทธิพิเศษ |
|-------|-------------------|-----------|
| Member | 0 บาท | ให้คะแนนพื้นฐาน |
| Silver | 5,000 บาท | ส่วนลด 3% |
| Gold | 15,000 บาท | ส่วนลด 5% + คะแนน x1.5 |
| Platinum | 50,000 บาท | ส่วนลด 8% + คะแนน x2 |
| VIP | 100,000 บาท | ส่วนลด 10% + คะแนน x3 + ฟรี delivery |

### อัปเกรดอัตโนมัติ
- ระบบเช็คยอดสะสมทุกครั้งที่ขาย
- ถ้าถึงเกณฑ์ → อัประดับทันที + แจ้ง LINE

## 📱 เชื่อม LINE OA

### สิ่งที่ต้องเตรียม
1. **LINE Official Account** (สมัครฟรีที่ [LINE OA Manager](https://manager.line.biz))
2. **LINE Developers Account** (สมัครที่ [developers.line.biz](https://developers.line.biz))
3. เปิดใช้ **Messaging API**

### ขั้นตอนตั้งค่า LINE

#### Step 1: สร้าง LINE Official Account
1. ไปที่ [LINE OA Manager](https://manager.line.biz)
2. กด **"สร้างบัญชี"** → กรอกข้อมูลร้าน
3. เลือกแผน (ฟรี = 500 ข้อความ/เดือน, Light = 5,000)

#### Step 2: เปิด Messaging API
1. ใน LINE OA Manager → Settings → Messaging API
2. กด **"เปิดใช้ Messaging API"**
3. เลือก Provider → สร้าง Channel

#### Step 3: ได้ Credentials
จาก [LINE Developers Console](https://developers.line.biz/console):
- **Channel Access Token** (long-lived) → copy
- **Channel Secret** → copy

#### Step 4: ตั้งค่าใน POS
1. ไปที่ **ตั้งค่า > LINE OA**
2. วาง **Channel Access Token**
3. วาง **Channel Secret**
4. กด **"ทดสอบการเชื่อมต่อ"** → ถ้าสำเร็จจะแสดง "เชื่อมต่อ LINE OA แล้ว ✓"

#### Step 5: ตั้งค่า Webhook
ใน LINE Developers Console:
- Webhook URL: `https://your-api.com/api/v1/line/webhook`
- เปิด **Use webhook** = ON
- ปิด **Auto-reply messages** = OFF (ให้ POS จัดการเอง)

#### Step 6: สร้าง LIFF (สำหรับ member linking)
1. ใน LINE Developers Console → LIFF tab → Add
2. ตั้ง Endpoint URL: `https://your-app.com/liff/register`
3. Scope: `profile`, `openid`
4. copy **LIFF ID** → ใส่ใน env

## ลูกค้าลิงก์ LINE กับสมาชิก

### วิธีที่ 1: ลูกค้าลงทะเบียนผ่าน LIFF
1. ลูกค้า Add LINE OA ของร้าน
2. LINE OA ส่งข้อความ "ลงทะเบียนสมาชิก" พร้อม link LIFF
3. ลูกค้ากดเปิด → กรอกเบอร์โทร → ระบบ match กับสมาชิกในร้าน
4. ลิงก์สำเร็จ → ส่งข้อความ/คูปอง/แจ้งคะแนนได้

### วิธีที่ 2: พนักงานลิงก์ให้
1. หน้าสมาชิก → เลือกสมาชิก → กด "ลิงก์ LINE"
2. แสดง QR code ให้ลูกค้าสแกน
3. ลูกค้า Add friend + ยืนยัน → ลิงก์สำเร็จ

## ส่งข้อความ/คูปองผ่าน LINE

### ส่งหาคนเดียว (Push Message)
1. ไปที่ **สมาชิก** → เลือกคน → กด **"ส่ง LINE"**
2. เลือกประเภท:
   - ข้อความ (text)
   - คูปอง (สร้าง Flex Message สวยงามอัตโนมัติ)
   - รูปภาพ
3. กด **"ส่ง"**

### Broadcast (ส่งหาทุกคน / กลุ่มเป้าหมาย)
1. ไปที่ **Communication > Broadcast**
2. เลือกกลุ่มเป้าหมาย:
   - ทุกคน
   - เฉพาะระดับ (เช่น Gold ขึ้นไป)
   - เฉพาะ Segment (เช่น "ไม่ซื้อ 30 วัน")
3. สร้างข้อความ/คูปอง
4. เลือก: ส่งทันที หรือ ตั้งเวลา
5. กด **"ส่ง"**

### ส่งคูปองอัตโนมัติ
ตั้งค่าใน **โปรโมชั่น > สมาชิก** ให้ส่งคูปองอัตโนมัติเมื่อ:
- สมัครสมาชิกใหม่ → ส่ง Welcome Coupon
- วันเกิด → ส่ง Birthday Coupon
- เลื่อนระดับ → ส่ง Upgrade Reward
- ไม่ซื้อ 30 วัน → ส่ง Win-back Coupon

## คำถามที่พบบ่อย

**Q: ลูกค้าไม่มี LINE ได้ไหม?**
A: ได้ ระบบ CRM ทำงานได้โดยไม่ต้อง LINE (ใช้เบอร์โทรเป็นหลัก) LINE เป็นช่องทางเสริม

**Q: ส่ง LINE ฟรีไหม?**
A: LINE OA ฟรี 500 ข้อความ/เดือน (Broadcast) ถ้าเกินต้อง upgrade plan (เริ่ม ~800 บาท/เดือน)

**Q: Push Message (ส่งหาคนเดียว) ฟรีไหม?**
A: Push Message ไม่นับ quota — ส่งได้ไม่จำกัด (ฟรี)
