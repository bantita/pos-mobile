# 05 — โปรโมชั่น & แคมเปญ CRM

## ประเภทโปรโมชั่นที่รองรับ

| ประเภท | คำอธิบาย | ตัวอย่าง |
|--------|----------|----------|
| **ส่วนลด %** | ลดราคาเป็นเปอร์เซ็นต์ | ลด 20% ทั้งบิล |
| **ส่วนลดเงิน** | ลดเป็นจำนวนเงิน | ลด 50 บาท เมื่อซื้อ 500+ |
| **คูปอง** | รหัสคูปองลดราคา | SAVE20 ลด 20% |
| **ราคาสมาชิก** | ราคาพิเศษเฉพาะสมาชิก | สมาชิก Gold ซื้อกาแฟ 35 บาท (ปกติ 45) |
| **ซื้อ X แถม Y** | ซื้อครบแถมฟรี | ซื้อ 2 แถม 1 |
| **Mix & Match** | ซื้อสินค้ากลุ่ม ได้ราคาพิเศษ | ซื้อ 3 ชิ้นจากกลุ่ม ลด 15% |
| **Happy Hour** | ลดเฉพาะช่วงเวลา | 14:00-16:00 ลด 30% เครื่องดื่ม |

---

## วิธีสร้างโปรโมชั่น

1. ไปที่ Tab **"โปรโมชั่น"** → กด **"+ สร้างโปรโมชั่น"**
2. เลือกประเภท
3. กรอกรายละเอียด:
   - ชื่อโปร
   - รหัสโปร (auto-generate หรือกำหนดเอง)
   - ช่วงวันที่ (เริ่ม-สิ้นสุด)
   - เงื่อนไข (ยอดขั้นต่ำ, สินค้าที่ร่วม, วันที่ใช้ได้)
   - รางวัล (ส่วนลด/สินค้าแถม/คะแนนพิเศษ)
4. กด **"บันทึก"** → โปรเริ่มทำงานอัตโนมัติ

## วิธีตั้งค่าโปรโมชั่นแต่ละประเภท

### ส่วนลด % / เงิน
1. โปรโมชั่น > "+ สร้าง" > "ส่วนลด %" หรือ "ส่วนลดเงิน"
2. ตั้งค่า:
   - สินค้าที่ร่วม (ทั้งหมด / เลือกหมวด / เลือกรายการ)
   - ยอดขั้นต่ำ
   - จำกัดสิทธิ์/วัน
3. ตั้งวันที่ เริ่ม-สิ้นสุด
4. บันทึก → ใช้ได้ทันทีตามวันที่ตั้ง

### คูปอง
1. โปรโมชั่น > "+ สร้าง" > "คูปอง"
2. กำหนดรหัส (เช่น `SAVE20`) หรือให้ระบบ auto-generate
3. ตั้งจำนวนสิทธิ์ (เช่น 100 ใบ)
4. ตั้งเงื่อนไข (ยอดขั้นต่ำ, สมาชิกเท่านั้น, วันที่)
5. ส่งผ่าน LINE / แจก ณ จุดขาย / พิมพ์ QR

### ซื้อ X แถม Y
1. โปรโมชั่น > "+ สร้าง" > "ซื้อ X แถม Y"
2. เลือกสินค้า X (ที่ต้องซื้อ) + จำนวน
3. เลือกสินค้า Y (ที่แถม) + จำนวน
4. ตั้งจำนวนสิทธิ์ / เงื่อนไขเพิ่ม
5. บันทึก

### Happy Hour
1. โปรโมชั่น > "+ สร้าง" > "Happy Hour"
2. เลือกสินค้า/หมวด ที่ร่วม
3. ตั้งเวลา (เริ่ม-สิ้นสุด) + วันในสัปดาห์
4. ตั้ง % ส่วนลด
5. ระบบ apply อัตโนมัติเมื่อถึงเวลา

## การใช้ร่วมกัน (Stacking Rules)

- ตั้งค่าได้ว่าโปรไหน **ใช้ร่วมกันได้** หรือ **ห้ามใช้ร่วม**
- ระบบจะเลือกโปรที่ **ให้ส่วนลดมากที่สุด** อัตโนมัติ (ถ้าห้าม stack)
- คูปอง + โปรร้าน → ปกติห้ามใช้ร่วม (ตั้งค่าได้)
- ราคาสมาชิก + คะแนน → ใช้ร่วมได้

## Dashboard โปรโมชั่น

ดูภาพรวม:
- จำนวนโปรที่ active
- จำนวนครั้งที่ใช้ (usage count)
- ยอดส่วนลดที่ให้ไป (total discount given)
- ROI ของแต่ละ campaign (ยอดขายที่เพิ่มขึ้น vs ส่วนลดที่ให้)
