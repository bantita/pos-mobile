# 07 — สินค้า & คลังสินค้า

## จัดการสินค้า

### เพิ่มสินค้า
1. Tab **"สินค้า"** → กด **"+ เพิ่มสินค้า"**
2. กรอก:
   - รหัสสินค้า (SKU) — auto-generate หรือกำหนดเอง
   - บาร์โค้ด (สแกนหรือพิมพ์)
   - ชื่อสินค้า
   - หมวดหมู่
   - ราคาขาย / ราคาทุน
   - หน่วยนับ (ชิ้น/ขวด/กล่อง ฯลฯ)
   - จำนวนเตือนขั้นต่ำ (min stock)
   - รูปภาพ (ถ่ายหรืออัปโหลด)
3. กด **"บันทึก"**

### หน่วยนับหลายระดับ (UOM)
ตัวอย่าง: น้ำดื่ม ขายได้ทั้ง "ขวด" และ "แพ็ค (12 ขวด)"

1. เข้าสินค้า → Tab **"หน่วยนับ"**
2. กด "+ เพิ่มหน่วย":
   - ชื่อหน่วย: แพ็ค
   - อัตราส่วน: 12 (1 แพ็ค = 12 ขวด)
   - บาร์โค้ดแพ็ค: (ถ้ามี)
   - ราคาขายแพ็ค: 160 บาท
   - ราคาทุนแพ็ค: 88 บาท
3. ตอนขาย สามารถเลือกขายเป็น "ขวด" หรือ "แพ็ค" ได้

### หมวดหมู่
- สร้างหมวดใหม่ (ชื่อ + สีประจำหมวด)
- ลากจัดลำดับ
- ลบ/แก้ไข

### Import สินค้า (CSV)
1. ดาวน์โหลด template CSV
2. กรอกข้อมูลใน Excel/Google Sheets
3. Export เป็น CSV → อัปโหลด
4. ระบบ preview → กด "Import"

## คลังสินค้า

### คลัง (Warehouse)
- **คลังหลัก** — เก็บสินค้า stock หลัก
- **คลังสาขา** — สต๊อกแต่ละสาขา
- **จุดขาย** — สต๊อก ณ POS (ถ้าแยก)

### ดูสต๊อก
Tab **"คลัง"** → แสดง:
- รายการสินค้าทั้งหมด + จำนวนคงเหลือ
- สถานะ: ปกติ 🟢 / ใกล้หมด 🟡 / หมดสต๊อก 🔴
- มูลค่าสต๊อกรวม

---

## รับสินค้าเข้าคลัง (Receive)

1. คลัง → **"รับสินค้า"** → กด **"+ สร้างใบรับ"**
2. เลือก:
   - คลังที่รับ
   - Supplier (ถ้ามี)
3. เพิ่มสินค้า:
   - สแกนบาร์โค้ด (ทีละชิ้น — auto count)
   - หรือ ค้นหา + กรอกจำนวน
4. กรอกราคาทุน (ถ้าต่างจากปกติ)
5. เลือก:
   - **"บันทึกแบบร่าง"** — ยังไม่เข้าสต๊อก (แก้ไขได้)
   - **"ยืนยันรับสินค้า"** — เข้าสต๊อกทันที + บันทึกต้นทุน

---

## เบิกสินค้าออก (Issue / Transfer)

1. คลัง → **"เบิกสินค้า"** → กด **"+ สร้างใบเบิก"**
2. เลือก:
   - คลังต้นทาง (เบิกจากไหน)
   - คลังปลายทาง (ส่งไปไหน)
3. เพิ่มสินค้า + จำนวน
   - ระบบเตือนถ้าสต๊อกไม่พอ
4. ยืนยัน → ตัดสต๊อกต้นทาง + เพิ่มปลายทาง

---

## Revise เอกสาร

ถ้าเอกสาร **ยืนยันแล้ว** แต่ข้อมูลผิด:
1. เปิดเอกสาร → กด **"Revise"**
2. กรอกเหตุผล (บังคับ)
3. ระบบสร้าง revision ใหม่เป็น draft
4. แก้ไข draft → ยืนยันอีกครั้ง
5. เอกสารเดิมเปลี่ยนสถานะเป็น "Revised"
6. ดู Revision History ได้ตลอด

---

## สแกนบาร์โค้ด (Inventory)

ใช้กล้องมือถือหรือ scanner:
- **รับสินค้า** — สแกนทีละชิ้น ระบบนับให้อัตโนมัติ
- **ตรวจนับสต๊อก** — สแกนเทียบกับระบบ
- **ค้นหาสินค้า** — สแกนเพื่อดูข้อมูล/สต๊อก

---

## แจ้งเตือนสต๊อก

ระบบแจ้งเตือนอัตโนมัติเมื่อ:
- สินค้า **ใกล้หมด** (ต่ำกว่า min stock)
- สินค้า **หมดสต๊อก** (= 0)
- แจ้งที่: Dashboard, Notification, LINE (ถ้าตั้ง)
