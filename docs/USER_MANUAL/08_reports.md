# 08 — รายงาน

## ภาพรวม

Tab **"รายงาน"** แสดงรายงาน 5 หมวดหลัก:

---

## 📊 รายงานยอดขาย

### Dashboard
- ยอดขายวันนี้ / สัปดาห์ / เดือน
- กราฟเปรียบเทียบ (วัน vs วัน, เดือน vs เดือน)
- ยอดขายแยกช่องทางชำระ (เงินสด / QR / บัตร)
- ยอดขายแยกพนักงาน

### รายงานรายวัน
- ตาราง: วันที่, ยอดขาย, จำนวนบิล, เฉลี่ย/บิล, กำไร, Margin
- filter: ช่วงวันที่, สาขา, POS
- Export: Excel / PDF

### รายงานพนักงาน
- ยอดขายแต่ละคน
- จำนวนบิล
- เฉลี่ยต่อบิล
- ranking

---

## 💰 รายงานกำไร

### Gross Profit
- กำไรขั้นต้น = ยอดขาย - ต้นทุน
- Margin % = กำไร / ยอดขาย × 100

### รายงานกำไรรายสินค้า
- สินค้าไหนกำไรดี / กำไรน้อย
- จัดอันดับ Top 10 กำไร

### Gauge Chart
- Margin เฉลี่ยของร้าน (เทียบ target)

---

## 📦 รายงานสินค้า

### Top Selling
- สินค้าขายดี Top 10/20/50
- แยกตาม: จำนวน, รายได้, กำไร

### สินค้าไม่เคลื่อนไหว (Dead Stock)
- ไม่มีการขาย > 30/60/90 วัน
- แนะนำทำโปรเพื่อระบาย

### หมวดหมู่
- ยอดขายแยกหมวด
- % contribution ของแต่ละหมวด

---

## 🏪 รายงานคลัง

### สต๊อกปัจจุบัน
- จำนวนคงเหลือแต่ละสินค้า
- มูลค่าสต๊อกรวม
- สินค้าใกล้หมด / หมดแล้ว

### Inventory Turnover
- อัตราการหมุนเวียนสินค้า
- สินค้าที่ turnover ต่ำ = Dead Stock

### ประวัติรับ/เบิก
- list เอกสารทั้งหมด
- filter: ประเภท, สถานะ, วันที่

---

## 🏢 รายงาน Enterprise

(สำหรับร้านที่มีหลายสาขา)

### เปรียบเทียบสาขา
- ยอดขาย, กำไร, Margin แต่ละสาขา
- ranking สาขา

### GMROI (Gross Margin Return on Investment)
- วัดผลตอบแทนจากเงินที่ลงทุนในสต๊อก
- GMROI = Gross Margin / Average Inventory Cost

### เปรียบเทียบ POS
- ยอดขายแต่ละจุดขาย
- ประสิทธิภาพพนักงาน

---

## Export รายงาน

ทุกรายงานสามารถ:
- **Export Excel** — download .xlsx
- **Export PDF** — download report พร้อมกราฟ
- **พิมพ์** — print ผ่าน browser
- **แชร์** — ส่งไฟล์ผ่าน LINE/Email

---

## ช่วงเวลาที่เลือกได้

| ตัวเลือก | คำอธิบาย |
|----------|----------|
| วันนี้ | เฉพาะวันนี้ |
| เมื่อวาน | วันก่อน |
| 7 วันล่าสุด | สัปดาห์ที่ผ่านมา |
| 30 วันล่าสุด | เดือนที่ผ่านมา |
| เดือนนี้ | 1 ถึงวันปัจจุบัน |
| เดือนก่อน | เดือนที่แล้วทั้งเดือน |
| กำหนดเอง | เลือก start - end date |
