# Xcellence ERP POS — Test Scenarios

## วิธีจดจำการทดสอบ

ทุก test case ระบุ:
- **ID**: รหัสอ้างอิง (TST-XXX)
- **Module**: โมดูลที่ทดสอบ
- **Precondition**: เงื่อนไขก่อนทดสอบ
- **Steps**: ขั้นตอนทดสอบ
- **Expected**: ผลลัพธ์ที่คาดหวัง
- **Status**: ⬜ ยังไม่ทดสอบ / ✅ ผ่าน / ❌ ไม่ผ่าน

---

## Module 1: สมัคร & เข้าสู่ระบบ

### TST-001: สมัครร้านค้าใหม่
- **Precondition**: ยังไม่มี account
- **Steps**:
  1. เปิดแอป → หน้า Welcome
  2. กด "สมัครร้านค้าใหม่"
  3. กรอกชื่อร้าน: "ร้านทดสอบ A"
  4. เลือกประเภท: "ร้านค้าปลีก"
  5. เลือกขนาด: "เล็ก"
  6. กรอกชื่อเจ้าของ, เบอร์โทร
  7. ตั้งรหัสผ่าน
  8. กด "สมัคร"
- **Expected**: เข้า Dashboard สำเร็จ, เห็นชื่อร้าน "ร้านทดสอบ A"
- **Status**: ⬜

### TST-002: เข้าสู่ระบบ
- **Steps**:
  1. เปิดแอป → หน้า Welcome
  2. กรอก Username + Password
  3. กด "เข้าสู่ระบบ"
- **Expected**: เข้า Dashboard, แสดง KPI ยอดขาย
- **Status**: ⬜

### TST-003: เข้าสู่ระบบ — รหัสผ่านผิด
- **Steps**: กรอก password ผิด → กด Login
- **Expected**: แสดง error "รหัสผ่านไม่ถูกต้อง", ไม่เข้า Dashboard
- **Status**: ⬜

## Module 2: POS ขายสินค้า

### TST-010: ขายสินค้าปกติ (เงินสด)
- **Precondition**: มีสินค้าในระบบ
- **Steps**:
  1. เข้าหน้า POS
  2. กดเลือกสินค้า "น้ำดื่มสิงห์" 3 ชิ้น
  3. กดเลือก "ขนมปังกรอบ" 2 ชิ้น
  4. ตรวจ subtotal = (10×3) + (25×2) = 80 บาท
  5. กด "ชำระเงิน" → เลือก "เงินสด"
  6. ใส่รับเงิน 100 บาท
  7. กด "ยืนยัน"
- **Expected**: 
  - ทอน 20 บาท
  - แสดงบิลสำเร็จ
  - ตะกร้าว่าง
  - ประวัติขายมีบิลใหม่
- **Status**: ⬜

### TST-011: ขายสินค้า + ส่วนลดรายการ
- **Steps**:
  1. เพิ่มสินค้า ฿89 จำนวน 1
  2. กดส่วนลดที่สินค้า → ใส่ 10%
  3. ตรวจราคาหลังลด = 89 - 8.9 = 80.1 บาท
  4. ชำระ
- **Expected**: ยอดสุทธิ = ฿80.10 (ปัดตามกฎ)
- **Status**: ⬜

### TST-012: ขายสินค้า + เลือกสมาชิก + สะสมคะแนน
- **Steps**:
  1. เพิ่มสินค้า ฿500
  2. กด "เลือกสมาชิก" → ค้นหา "สมชาย" → เลือก
  3. ชำระ QR Code
  4. ตรวจคะแนนที่สมาชิกได้รับ
- **Expected**: 
  - แสดงข้อมูลสมาชิก (แต้ม, ยอดสะสม, คูปอง)
  - หลังชำระ: คะแนนเพิ่ม (500 ÷ earnRate)
- **Status**: ⬜

### TST-013: ขายสินค้า + ใช้คูปอง
- **Steps**:
  1. เพิ่มสินค้า ฿400
  2. กรอกคูปอง "SUMMER2024"
  3. กด "ใช้คูปอง"
- **Expected**: 
  - ยอดขั้นต่ำ 300 → ผ่าน
  - ลด 100 บาท → ยอดสุทธิ ฿300
- **Status**: ⬜

### TST-014: ขายสินค้า + ใช้คะแนน
- **Steps**:
  1. เลือกสมาชิกที่มีคะแนน 200
  2. เพิ่มสินค้า ฿300
  3. กด "ใช้คะแนน" → ใส่ 100 คะแนน
  4. ชำระ
- **Expected**: ส่วนลดจากคะแนน = 100 × redeemRate
- **Status**: ⬜

### TST-015: พักบิล (Hold) + เรียกคืน
- **Steps**:
  1. เพิ่มสินค้า 3 รายการ
  2. กด "พักบิล"
  3. เปิดบิลใหม่ → ขายสินค้าอื่น → ชำระ
  4. กด "เรียกบิลพัก"
  5. เลือกบิลที่พักไว้
- **Expected**: สินค้า 3 รายการกลับมาในตะกร้า
- **Status**: ⬜

### TST-016: ยกเลิกบิล (Void)
- **Steps**:
  1. ไปหน้าประวัติขาย
  2. เลือกบิลที่ต้องการยกเลิก
  3. กด "คืนสินค้า/ยกเลิก"
- **Expected**: สถานะบิลเปลี่ยนเป็น "ยกเลิก"
- **Status**: ⬜

## Module 3: สินค้า

### TST-020: เพิ่มสินค้าใหม่
- **Steps**:
  1. เข้าหน้า สินค้า → tab รายการ
  2. กด "+ เพิ่มสินค้า"
  3. กรอก: รหัส P100, ชื่อ "สินค้าทดสอบ", หมวดหมู่ "ของใช้"
  4. ราคาทุน 50, ราคาขาย 100
  5. บันทึก
- **Expected**: สินค้าแสดงในรายการ, กำไร 100%
- **Status**: ⬜

### TST-021: แก้ไขราคาสินค้า
- **Steps**: กดแก้ไข → เปลี่ยนราคาขาย → บันทึก
- **Expected**: ราคาใหม่แสดงในรายการ + tab กำหนดราคา
- **Status**: ⬜

### TST-022: ดูสต๊อก (tab คลังสินค้า)
- **Steps**: เข้า tab "คลังสินค้า"
- **Expected**: เห็น KPI (สินค้าทั้งหมด, ปกติ, ใกล้หมด, หมดสต๊อก) + ตาราง
- **Status**: ⬜

## Module 4: CRM

## Module 5: โปรโมชั่น

### TST-040: สร้างโปรลดเปอร์เซ็นต์
- **Steps**:
  1. โปรโมชั่น → สร้างใหม่ → เลือก "ลด %"
  2. กรอก: ลด 10%, ทุกสินค้า, วันที่ 1-30 มิ.ย.
  3. บันทึก
- **Expected**: โปรแสดงในรายการ สถานะ Active
- **Status**: ⬜

### TST-041: โปรคำนวณอัตโนมัติเมื่อขาย
- **Steps**:
  1. มีโปร "ลด 10% เครื่องดื่ม" Active
  2. เข้า POS → เพิ่มสินค้าหมวด "เครื่องดื่ม"
  3. ตรวจราคาที่แสดง
- **Expected**: ลด 10% อัตโนมัติ
- **Status**: ⬜
