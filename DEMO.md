## 🏪 ประเภทร้านค้า (Store Type)

ระบบรองรับ 3 ประเภท เลือกได้ตั้งแต่หน้าสมัคร หรือเปลี่ยนที่ตั้งค่าร้าน:

| ประเภท | Icon | คำอธิบาย | ฟีเจอร์เด่น |
|---|---|---|---|
| **SERVICE** | ✂️ | ร้านตัดผม, สปา, นวด | เลือกช่าง/พนักงานบริการทุกบิล, SC |
| **RETAIL** | 🏪 | ร้านค้าปลีก/ค้าส่ง | หลายจุดขาย POS, คลังสินค้า |
| **ENTERPRISE** | 🏢 | หลายสาขา | หลายสาขา + หลายจุดขาย, Sync |

### Service Charge (เซอร์วิสชาร์จ)

```
ตั้งค่าที่: ตั้งค่า → ตั้งค่าร้านค้า → เซอร์วิสชาร์จ
โหมด:     เปอร์เซ็นต์ (%) หรือ จำนวนคงที่ (฿)
ตัวอย่าง: 10% → ยอด ฿1,000 บวก SC ฿100 = ฿1,100
```

## 👥 จัดการพนักงานและผู้ใช้งาน

### หลักการ
- **พนักงาน (Employee)** ไม่จำเป็นต้องมี User Account
- **ผู้ใช้งาน (User)** ต้องเชื่อมกับ Employee เสมอ
- ไปที่: ตั้งค่า → จัดการพนักงาน / จัดการผู้ใช้งาน

### พนักงาน Demo (7 คน)

| รหัส | ชื่อ | ตำแหน่ง | แผนก | ช่าง? | มี User? |
|---|---|---|---|---|---|
| EMP001 | สมชาย ใจดี | เจ้าของ | บริหาร | ❌ | ✅ Owner |
| EMP002 | สมหญิง จริงใจ | ผู้จัดการ | บริหาร | ❌ | ✅ Manager |
| EMP003 | สมศักดิ์ ขยัน | แคชเชียร์ | ขาย | ❌ | ✅ Cashier |
| EMP004 | ประเสริฐ มั่นคง | พนักงานคลัง | คลังสินค้า | ❌ | ✅ Stock Staff |

### ข้อมูลพนักงานที่บันทึก
1. **ข้อมูลส่วนบุคคล** — ชื่อ-สกุล, บัตร ปชช., วันเกิด, สัญชาติ, ที่อยู่
2. **ช่องทางติดต่อ** — โทร, อีเมล, Line ID, บุคคลฉุกเฉิน
3. **เอกสาร** — สำเนาบัตร, ทะเบียนบ้าน, รูปถ่าย, PDPA
4. **การจ้างงาน** — ตำแหน่ง, แผนก, วันเริ่มงาน, ประเภทสัญญา

## 🎮 Demo Scenarios ใหม่

### Scenario 6 — ร้านบริการ (Service Store)
```
1. หน้าสมัคร → เลือกประเภท "ร้านบริการ"
2. หรือ: ตั้งค่า → ตั้งค่าร้านค้า → เปลี่ยนเป็น SERVICE
3. ตั้งค่า SC: เปิด → เปอร์เซ็นต์ → 10%
4. Tab ขาย → กดสินค้า "ตัดผมชาย" (สีม่วง บริการ)
5. Popup เลือกช่าง → เลือก "ช่างเอ (สมชาย)"
6. กด "ตัดผมชาย" อีกครั้ง → เลือก "ช่างบี (สมหญิง)"
7. ดูตะกร้า → เห็น 2 บรรทัด: ชื่อสินค้า + ชื่อช่าง
```

### Scenario 7 — จัดการพนักงาน
```
1. ตั้งค่า → จัดการพนักงาน
2. ดูรายชื่อ 7 คน (ค้นหา "ป๊อป" → พบ EMP005)
3. กด EMP005 → แก้ไข → เปลี่ยนตำแหน่งเป็น "หัวหน้าช่าง"
4. กด "+ เพิ่ม" → กรอกข้อมูลพนักงานใหม่
5. เปิด toggle "เป็นช่างบริการ" → บันทึก
6. กลับไปหน้า POS → กดสินค้าบริการ → เห็นช่างใหม่ใน popup
```

### Scenario 8 — จัดการผู้ใช้งาน
```
1. ตั้งค่า → จัดการผู้ใช้งาน
2. เห็น 5 users + role badge สี
3. กด "Pause" ที่ usr-003 → สถานะเปลี่ยนเป็น "ระงับ"
4. กด "+ เพิ่ม" → เลือกพนักงาน "นิดา แก้วใส" (ยังไม่มี user)
5. กรอก username: 0866666666 → เลือก role: cashier
6. บันทึก → นิดาเข้าใช้ระบบได้
```

### Scenario 9 — VAT & ประเภทสินค้า
```
1. Tab สินค้า → กด "+" เพิ่มสินค้าใหม่
2. Section ราคา: เปิด VAT → เลือก 7%
3. Section ประเภท: เลือก "สินค้าบริการ" (สีม่วง)
4. บันทึก → สินค้าแสดง badge "บริการ" ที่หน้า POS
```
